## Dasar Teori

Xinu adalah shell sederhana yang memungkinkan user untuk menjalankan perintah untuk berinteraksi dengan sistem. perintah tersebut digunakan untuk melakuikan pengujian fungsi kernel, melihat status kernel dll.

dan pada modul kali ini, kita akan mempelajari xinu melalui eksploarsi xinu, yang dimana kita akan mempelajari konsep dasar sistem operasi, kita sudah mengakses xinu pada pertemuan kemarin dikelas dan saat ini adalah saatnya untuk mengeksplorasi.


## Guided

1.  Perintah Xinu, setelah mengetik "help" di Terminal

![alt text](image.png)

Terdapat 23 command pada Xinu

command pertama adalah argecho

![alt text](image-1.png)

digunakan untuk menampilkan argumen dari pengguna

command kedua adalah arp

![alt text](image-2.png)

yang dimana fungsinya adalah untuk menampilkan ip adress

command cat
digunakan untuk membaca file, karena saya tidak memasukan nama file yang akan dibaca jadinya tidak muncul apa apa

command clear
untuk membersihkan terminal dari perintah perintah yang sudah dijalankan

command date

![alt text](image-3.png)

untuk menampilkan tanggal, hari, dan waktu

command devdump

![alt text](image-4.png)

digunakan untuk menampilkan informasi tabel device yang ada pada sistem

command echo

![alt text](image-5.png)

digunakan untuk mendisplay text, fungsi tepatnya adalah untuk menampilkan huruf yang kita keik, jika hanya menulis echo makan tidak akan muncul apa apa.

command help

![alt text](image-6.png)

berfungsi untuk menampilkan jenis jenis command yang tersedia pada xinu.

command ls
digunakan untuk menampilkan direktori yang ada

command kill
digunakan untuk memberhentikan suatu proses

command memdump

![alt text](image-8.png)

digunakan untuk melihat isi dari fisik ram pada alamat tertentu secara langsung. sebelum menggunakannya kita harus menggunakan memstat dulu untuk melihat adress dan lenghtmya


command memstat 

![alt text](image-7.png)

digunakan untuk menampilkan alamat memory yang digunakan pada sistem xinu

command ns

![alt text](image-9.png)

ns atau nameserver digunakan untuk melihat informasi server yang digunakan oleh sistem jaringan

command netinfo

![alt text](image-10.png)

digunakan untuk melihat ip adressdari device kita, termasuk IP router, server DNS dll

command ping

![alt text](image-11.png)

digunakan untuk melihat ping, sama seperti di cmd pada windows

command ps

![alt text](image-12.png)

digunakan untuk melihat data proses pada sistem

command sleep
digunakan untuk memberhentikan secara sementara eksekusi proses selama waktu yang digunakan

command tee
digunakan untuk memabca input dari pengguna

command udp
digunkaan untuk mengirim paket data dan menggunakan protokol UDP pada jaringan, untuk yang udp ini harus ada device lain yang menggunakan jaringan yamg sama. contoh ppenggunaan ada dibawah

![alt text](image-13.png)

command udpecho

![alt text](image-14.png)

digunakan untuk mengetest kembali alat uji coba untuk memastikan bawha protokol jaringan, khususnya udp itu berfungsi dengan benar

command udpserver
digunakan untuk menjalankan server udp di xinu yang siap menerima paket data dari klien (melalui IP dan port tertentu)

command uptime

![alt text](image-15.png)

digunakan untuk mleihat sudah berapa lama user menggunakan xinu

command ?
sama seperti help, digunakan untuk menampilkan list command yang dapat digunakan di terminal
